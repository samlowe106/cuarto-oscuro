# cuarto-oscuro

From the Spanish term for "black room", cuarto-oscuro is a command line program that converts image files to another image format.

## Usage
```
Usage: cuarto-oscuro [OPTIONS] <PATH>

Arguments:
  <PATH>  The input folder path

Options:
  -i, --input-formats <INPUT_FORMATS>  Input image formats [default: jpg jpeg png bmp tiff]
  -o, --out-format <OUT_FORMAT>        Target format [default: webp]
  -h, --help                           Print help
  ```
